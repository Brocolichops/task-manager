# task-manager
Programming Principles Group Project.
This is to-do list built purely on C. 
It allows the user to add and monitor tasks. Each task has the following attributes:
  - ID
  - Description/Name
  - Priority
  - Status\n
The application is text-based and works on the CLI.
Users have the ability to add tasks, remove tasks, update task information, filter tasks, and mark tasks as completed.
